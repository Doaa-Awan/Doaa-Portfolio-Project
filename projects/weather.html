Weather App

Try it out

Made with HTML, CSS, JavaScript, and Bootstrap. This app was created using VS Code, and hosted on Netlify (using GitHub).

mobile

Responsive weather app that uses real live data from Open Weather Map API. The app features a dark-mode toggle switch, current location button, and a refresh page button.

desktop

The desktop version includes animated CSS background, video backgrounds, and audio toggle button (for video). There are theme buttons to switch between the backgrounds. There is also a light button that changes the opacity of the white sections from semi-transparent to fully opaque, to make it easier to see the information on a darker background.

wireframe

I tried to create a layout that was user-friendly and that also had a dark mode toggle switch. The final product for the mobile and desktop versions of the app were very similar to the original design.
